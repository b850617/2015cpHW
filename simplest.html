筆記

1. 如果從學校電腦抓12G的檔案 , leo 整個套件要自己去下載 ,安裝到 site- package 裡面。


2. launchLeo.py 和 start 可以從老師提供的壓縮檔中找到 , 複製到相同路徑即可 ( start 多個啟動 程序 )。


3. 檢查分支 : git bramch

新建分支 : git branch (分支名稱)

切換分支 : git checkout (分支名稱)


4. 因所在的分支不同 , 資料夾所顯示的檔案也有所不同

5. 新建的分支會複製原有的分支檔案


6. dir : 檢察所屬分支的檔案數量和狀況


7. git config (--global) user.name .... , 括弧內若省略 , 此次僅為單 一1次 , 若沒有省略 , 則以後皆為默認


8. REM = REMARK , .bat 檔 = batch 批次檔 , -s = set 設定


9. 儲存網內內容的 .leo 存檔之後所產生的simplest.html檔案 可以很方便的檢查近端leo的編輯狀況

10. 持續更新中
